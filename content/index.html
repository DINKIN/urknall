Urknall Documentation

Urknall is a tool to create project specific provisioning tools. It provides mechanisms to execute commands on a host. This is the main intent of the library. For rapid prototyping there is a binary to scaffold a basic application and add some basic templates we provide. The files created should be considered as starting point and not the absolute truth.
